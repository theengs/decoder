# Supporting the project

If you like the project and/or used it please consider supporting it! It can be done in different ways:

  • Helping others in the community (opens new window)
  • Contribute to the code (opens new window) or the documentation,
  • Buy devices, boards or parts from the OpenMQTTGateway compatible web site (opens new window), the devices and parts linked use affiliated links,
  • Donate or sponsor the project developers (opens new window)
  • Make a video or a blog article about what you have done with TheengsDecoder (opens new window) and share it.
